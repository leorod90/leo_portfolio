Forkify

View Project View Code

This is project is called the forkify app and uses the Food2Fork api. While the design may be simple there is alot going on behind the scene. You can enter any food the you want and an array of recipes will be returned. From there you can pick one of the items you like to see the recipe. There is also a like function that will save any recipes you like. Even if the page is refreshed!
